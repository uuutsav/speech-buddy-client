
# Setting Up

- Make parent repository
```
mkdir speech-buddy
```
```
cd speech-buddy
```
- Clone client file
```
git clone https://github.com/uuutsav/speech-buddy-client.git
```
- Check out server repository
 [Server Repo](https://github.com/uuutsav/speech-buddy-server).


```
cd speech-buddy-client
npm i
npm run start
```
